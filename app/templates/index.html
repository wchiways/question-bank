快速开始

几分钟即可完成部署

1

克隆项目

git clone https://github.com/wchiways/question-bank.git
2

安装依赖

cd ocs-tiku && uv sync
3

配置AI服务

cp config.example.json config.json

编辑 config.json，填入你的AI服务API密钥

4

启动服务

uv run uvicorn app.main:app --reload

访问 http://localhost:8000/docs 查看API文档
